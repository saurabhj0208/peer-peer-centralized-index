Before executing the following steps, make sure all RFC files are in same directory as the client.py 

STEPS:-
1. Open command prompt and go the directory of 'server.py' file.
2. Execute the following command:	python server.py
3. Open another command prompt and go the directory of 'client.py' file.
4. Execute the following command:	python client.py
5. Enter any decimal port number when prompted.
6. After that, provide the inputs according to your preferences and perform the desired operations.
